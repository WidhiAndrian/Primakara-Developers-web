Java Script

Java Script adalah bahasa pemrograman yang digunakan dalam pengembangan website agar lebih dinamis dan interaktif.

CSS

CSS adalah sebuah bahasa sederhana yang digunakan untuk menambahkan gaya/styling ke sebuah halaman website.

HTML

HTML adalah bahasa yang digunakan membuat dan mengatur elemen seperti judul, paragraf, halaman web.

React.js

React.js libray JavaScript yang digunakan untuk membangun user interface yang interaktif berbasis component.

Vue.js

Vue.js adalah libray JavaScript bersifat progresif, bersumber terbuka untuk membangun antarmuka pengguna.

Express.js

Express.js adalah framework web app untuk Node.js yang ditulis dengan bahasa pemrograman JavaScript.
